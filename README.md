# AvtoHelp — Car Towing Service in Kyiv and Kyiv Region

**AvtoHelp** is a modern website for a car towing service operating 24/7 in Kyiv and the Kyiv region. The project is designed for quick and convenient dispatch of specialized towing equipment in any roadside situation.

## 🚀 Key Features

- **Responsive Design**: Flawless display on all devices — from smartphones to desktops.
- **Quick Call**: Direct call buttons and integration with popular messengers (Viber, Telegram, WhatsApp).
- **SEO Optimized**: Dedicated pages for Kyiv districts and regional cities for better search engine visibility.
- **24/7 Service**: Emphasis on round-the-clock support and availability.
- **Transparent Pricing**: Convenient price table and clear service descriptions.

## 🛠 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Components**: UI components using `swiper` for the gallery
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📦 Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/helen-akateva/avtohelp.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
